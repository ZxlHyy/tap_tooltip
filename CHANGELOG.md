## 0.0.5

- Expanded package documentation for pub.dev with installation, usage,
  controller, styling, positioning, dismissal, and example sections.
- Documented all public tooltip content APIs: `tooltip`, `tooltipWidget`, and
  `tooltipBuilder`.
- Documented safety behavior, controller-only interaction, and tooltip styling
  options.

## 0.0.2

Initial release.

- Added `TapTooltip`, a click-triggered Flutter tooltip widget.
- Added `TapTooltipController` for programmatic show, hide, and toggle flows.
- Added support for top, bottom, left, and right tooltip placement.
- Added text, widget, and builder-based tooltip content APIs.
- Added customizable bubble decoration, text style, padding, constraints, size,
  arrow dimensions, and arrow radius.
- Added automatic screen-boundary detection with configurable safe padding.
- Added optional outside-tap dismissal, controller-only interaction, and
  background blur support.
